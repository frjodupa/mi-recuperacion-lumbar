# PROJECT DNA - Mi Recuperación Lumbar

## Visión

Esta aplicación no es un gestor de ejercicios.

Es un asistente inteligente especializado en la recuperación tras una artrodesis lumbar.

Cada pantalla debe ayudar al paciente a sentirse acompañado, seguro y motivado.

La aplicación debe ser extremadamente sencilla de utilizar incluso para personas mayores.

## Filosofía

Menos opciones.
Más claridad.

Menos datos.
Más utilidad.

Cada botón debe tener un propósito.

Nunca añadir funciones que compliquen la experiencia.

## Diseño

- Aspecto premium.
- Mucho espacio en blanco.
- Tarjetas grandes.
- Tipografía muy legible.
- Animaciones suaves.
- Interfaz tipo aplicación Apple.
- Responsive.
- Aprovechar el ancho de pantalla.

## IA

Toda la inteligencia gira alrededor del paciente.

La IA podrá:

- Leer informes médicos.
- Leer capturas de pantalla.
- Interpretar informes PDF.
- Detectar evolución.
- Adaptar la rehabilitación.
- Explicar informes con lenguaje sencillo.
- Recomendar ejercicios.

Nunca diagnosticar.

Siempre recomendar consultar al profesional.

## Prioridades

1. Onboarding excelente.
2. Rehabilitación.
3. Evolución.
4. IA.
5. Fitness.
6. Ajustes.

## Regla principal

Cada cambio debe hacer la aplicación más sencilla.

Nunca más compleja.
